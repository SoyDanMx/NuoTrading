### Frontend
- Located in the `frontend/` directory.
- Built with Next.js 14 (App Router) and TypeScript.
- Uses Tailwind CSS for styling.
- To run locally: `cd frontend && npm install && npm run dev`.

### Backend
- Located in the `backend/` directory.
- Built with FastAPI (Python).
- Uses Pandas for data analysis.
- To run locally: `cd backend && pip install -r requirements.txt && uvicorn main:app --reload`.

## ğŸ“ Commit Guidelines

We use [Conventional Commits](https://www.conventionalcommits.org/) for our commit messages:
- `feat:` for new features.
- `fix:` for bug fixes.
- `docs:` for documentation changes.
- `style:` for changes that do not affect the meaning of the code (formatting, missing semi-colons, etc).
- `refactor:` for code changes that neither fix a bug nor add a feature.
- `perf:` for code changes that improve performance.
- `test:` for adding missing tests or correcting existing tests.
- `chore:` for changes to the build process or auxiliary tools and libraries.
